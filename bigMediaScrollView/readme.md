## Big Media Scroll View

### Features
+ Responsive big image/video bakcground
+ Highly customizable

### How to Use

#### Menu Items
+ Add as many menu items as you like
+ Put target background id in "data-bg"
+ Put target background type (video os static) in "data-bgtype" to make the audio control work accordingly

#### Audio Control Button
+ Will mute or unmute all background music by default

#### Background Items
+ Change its image background in CSS files

#### Contents
+  Give an ID to every scrollview-item for triggers
